# Pi 網頁介面顏色主題範例

## 🌅 **暖色調主題（橙色系）**
```css
:root {
    --primary-color: #ff6b35;        /* 橙色 */
    --secondary-color: #f7931e;      /* 金橙色 */
    --background-color: #fff8f5;     /* 暖白色 */
    --surface-color: #ffffff;        
    --text-color: #2c1810;           /* 深棕色文字 */
    --text-secondary: #8b4513;       
}
```
